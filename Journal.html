This week we learned about Binary. Binary is a series of 1’s and 0’s that can create complex code. Similar to the decimal system each nuuber repersents a value, 1,2,4,8,16,32,64,ect. Through we don’t use binary very often it is important to understand the base of where are code comes from. We can use Binary to create images sounds and programs.
