Tech Tuesday

Feb 20
Raspberry pi’s were created in 2012 in the uk. Though computers are a widely used device, in many developing countries they are hard to access, this is where the raspberry pie comes in. They are a low costing single computer board around the size of a credit card. They were created to be widely accessible while still being affordable and useful. Raspberry pi’s are mini-computers that are able to exchange and process information from multiple different devices. They are often used to learn programming or for data management. A raspberry pie has similar features to a motherboard while taking out things that are not necessary such as its internal storage. To use a raspberry pi you'll need an sd card which contains an operating system, which is needed for the computer to work. Raspberry pie’s are compatible with the linux os which reduces the amount of storage required. After setting up the os to the raspberry pie you can connect the raspberry pie to monitors ,hdmis television, mice and keyboards as an output device.

Raspberry Pis have many features that make them a popular and very versatile tool. These include: CPU, HDMI and USB ports, GPU, Memory, an Ethernet port, an SD card, and GPIO pins. The CPU or Central Processing Unit acts as the main base of operations for the Raspberry pi and is used to perform actions using logical and mathematical operations. An HDMI (High Definition Multimedia Interface) is useful because it gives options for video output, and it has a USB port for connecting to other devices. The GPU or Graphic Processing Unit helps the Raspberry Pi to make quicker calculations. It has memory to store real-time information to be easily referred to later. It has an Ethernet port for wired internet access. A Secure Digital card is used instead as a storage device and is necessary for it to function, as it contains the operating system that turns it on and is used to store data. GPIO pins are General Purpose Input and Output pins that can be used to interact with other electronic circuits and can be programmed to read and control electronic signals from other boards. Finally, it has LEDs with different colors to show different things, like if it is powered on or to monitor Ethernet connection.
